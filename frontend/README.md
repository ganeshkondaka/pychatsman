# Frontend

This is the React frontend for the Python AI Assistant application.

## Technologies Used

- React 19
- Vite
- Tailwind CSS
- Axios
- React Icons

## Installation

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Development

To start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

## Build

To build for production:
```bash
npm run build
```
